# Coming Soon Project

This project is a simple "Coming Soon" webpage designed to inform visitors that the site is under construction. It features a black background and an alert that appears 5 seconds after the page is opened.

## Project Structure

```
coming-soon-project
├── html
│   └── index.html
├── css
│   └── styles.css
├── js
│   └── script.js
└── README.md
```

## Files Description

- **html/index.html**: Contains the basic structure of the "Coming Soon" page, linking to the CSS for styling and the JavaScript for functionality.
  
- **css/styles.css**: Styles the page with a black background and additional text or layout styles.

- **js/script.js**: Contains the JavaScript code that triggers an alert 5 seconds after the page is opened.

## Setup Instructions

1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the "Coming Soon" page.
3. The alert will automatically appear 5 seconds after the page loads.

Feel free to customize the styles and content as needed!
